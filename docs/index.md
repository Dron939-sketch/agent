# Фреди

Фреди — open-source платформа персонального AI-помощника:

- 🧠 Мульти-LLM мозг (Claude / OpenAI / DeepSeek / Ollama) с auto-router и fallback.
- 🛠️ Tool registry: web search, calculator, fetch URL, кастомные плагины.
- 🤖 Мульти-агент оркестратор (Planner → Researcher → Coder → Critic → Executor).
- 🧬 Долговременная векторная память + Varitype-профиль пользователя.
- 🎙️ Голос: Deepgram + Yandex SpeechKit STT, Yandex TTS.
- 🔔 Web Push уведомления (VAPID).
- 🎨 WOW-фронтенд: Next.js 14 + R3F 3D-аватар + Framer Motion + ⌘K.

См. [Quickstart](quickstart.md) и [Архитектуру](architecture.md).
